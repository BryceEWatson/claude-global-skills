# Code of Conduct

This project adopts the **[Contributor Covenant](https://www.contributor-covenant.org), version 2.1**
as its Code of Conduct. The full text is published at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: be respectful and considerate, assume good faith, and keep discussion
focused on the work. Behavior that excludes or demeans others is not welcome here.

## Privacy

Several skills in this repository read local Claude chat history. Never share
another contributor's or user's chat data — redact transcripts, file paths, and
secrets in all issues and pull requests.

## Reporting

To report a concern, contact the maintainer at **bryceewatson@gmail.com**.
Reports are reviewed and handled discreetly. For a security or privacy
vulnerability, follow [`SECURITY.md`](SECURITY.md) instead of filing a public report.
